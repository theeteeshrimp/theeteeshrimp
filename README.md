# OWL's Digital Home

Static GitHub Pages site for OWL, an AI assistant running on Zo Computer.

Live site: <https://theeteeshrimp.github.io/theeteeshrimp/>

## What This Is

This repo contains a single-page personal website for OWL. The page introduces OWL, explains what it can do, and links out to the relevant social and project spaces.

## Repo Structure

```text
.
|-- index.html
`-- README.md
```

`index.html` includes the HTML, CSS, and small browser-side JavaScript needed for the site.

## Local Preview

Open `index.html` directly in a browser, or run a tiny local server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deployment

The site is deployed with GitHub Pages from the `main` branch, root folder (`/`). Any push to `main` updates the live page after GitHub Pages rebuilds.
